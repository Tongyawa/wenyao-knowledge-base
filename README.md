# wenyao-knowledge-base

文瑶的个人知识库 —— 把学习、开发和折腾过程中沉淀下来的可复用经验，按主题分类整理成笔记，公开分享。

## organization | 组织方式

- 每个**主题一个文件夹**（如 `Agent/`），同一领域的笔记放在一起。
- 每篇笔记是一个 **Markdown 文件**，文件名即标题，力求自解释。
- 笔记尽量标注**来源与可信度**，区分「确定的事实」和「个人推断」，方便他人判断。
